## Observation 1 : La Négation au Passé

Regardez : "Je **n'**ai **pas** compris".

La négation (ne ... pas) se place autour de **l'Auxiliaire**.
> **Sujet + NE + Auxiliaire + PAS + Participe**

*   Ex: Je **n'**ai **pas** aimé.
*   Ex: Elle **n'**est **pas** venue.

### Pratique 1 : Les difficultés (Négation)
**Mettez les phrases à la forme négative.**

1.  J'ai trouvé un appartement. -> *Je n'ai pas trouvé d'appartement.*
2.  J'ai compris le problème. -> *Je...*
3.  Elle est partie. -> *Elle...*
4.  Nous avons fini. -> *Nous...*
5.  Tu as vu le film. -> *Tu...*

<details>
<summary>Correction</summary>

1.  Je **n'**ai **pas** trouvé...
2.  Je **n'**ai **pas** compris.
3.  Elle **n'**est **pas** partie.
4.  Nous **n'**avons **pas** fini.
5.  Tu **n'**as **pas** vu.
</details>

## Observation 2 : Les Adverbes (Bien, Mal, Beaucoup)

Regardez : "J'ai **bien** appris".

Les adverbes courts se placent **ENTRE** l'auxiliaire et le participe.
> **Auxiliaire + [Adverbe] + Participe**

*   J'ai **bien** dormi.
*   Il a **beaucoup** travaillé.
*   Tu as **trop** parlé.

### Pratique 2 : Les progrès (Adverbes)
**Insérez l'adverbe dans la phrase.**

1.  J'ai compris. (bien) -> *J'ai bien compris.*
2.  Il a aimé le chocolat. (beaucoup) -> *Il a...*
3.  Nous avons marché. (trop) -> ...
4.  Tu as dormi ? (bien) -> ...
5.  Elles ont travaillé. (mal) -> ...

<details>
<summary>Correction</summary>

1.  J'ai **bien** compris.
2.  Il a **beaucoup** aimé.
3.  Nous avons **trop** marché.
4.  Tu as **bien** dormi ?
5.  Elles ont **mal** travaillé.
</details>

### "L'Exagérateur" �xܲ

**Vous exagérez tout ! Utilisez "Trop" ou "Beaucoup".**

1.  J'ai mangé. -> *J'ai trop mangé !*
2.  J'ai aimé. -> *J'ai beaucoup aimé !*
3.  J'ai marché. -> ...
