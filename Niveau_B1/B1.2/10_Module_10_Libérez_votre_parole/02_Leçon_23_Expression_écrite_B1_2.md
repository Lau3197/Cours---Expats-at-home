# Leçon 23 : Expression écrite B1.2

**Niveau** : B1.2  
**Module** : Module 10: Libérez votre parole
**Type** : Compétences écrites  
**Durée estimée** : 2-3 heures  
**Objectifs** : Practice B1.2 writing skills

---

## B1.2 Writing Competencies

At B1.2 level, you should be able to:
- Write detailed opinions with arguments
- Write narratives with varied past tenses
- Write formal letters and emails
- Write reports and summaries

---

## Practice 1: Opinion essay (150 words)

**Topic:** "Le télétravail est-il une bonne chose ?"

**Structure:**
1. Introduction (present the topic)
2. Arguments pour (D'un côté... / Premièrement...)
3. Arguments contre (D'un autre côté... / Cependant...)
4. Your opinion (À mon avis...)
5. Conclusion (En conclusion...)

**Required elements:**
- At least 2 arguments per side
- Opinion connectors
- Cause and consequence connectors

---

## Practice 2: Narrative (150 words)

**Topic:** "Un voyage inoubliable"

**Required tenses:**
- Imparfait (setting, descriptions)
- Plus-que-parfait (background)
- Passé composé (events)

**Structure:**
- Setting (C'était... / Il faisait...)
- Background (J'avais préparé... / J'avais déjà...)
- Events (Soudain... / Ensuite...)
- Conclusion (Finalement...)

---

## Practice 3: Formal email (100 words)

**Situation:** Request for internship

**Required:**
- Formal greeting
- Clear request
- Qualifications
- Formal closing
