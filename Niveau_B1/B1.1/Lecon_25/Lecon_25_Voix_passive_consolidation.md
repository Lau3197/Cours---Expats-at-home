# Leçon 25 : Voix passive en contexte (Consolidation)

**Niveau** : B1.1  
**Type** : Consolidation  
**Durée estimée** : 2-3 heures  
**Objectifs** : Practice the passive voice in various contexts

---

## Review: Passive Voice

### Formation: ÊTRE + PARTICIPE PASSÉ

| Tense | Example |
|:------|:--------|
| Présent | Le rapport est rédigé |
| Passé composé | Le rapport a été rédigé |
| Imparfait | Le rapport était rédigé |
| Plus-que-parfait | Le rapport avait été rédigé |
| Futur | Le rapport sera rédigé |

---

## Practice 1: Quick transformation

**Transform to passive:**

1. Le chef approuve le projet. → _______________
2. L'équipe a terminé le travail. → _______________
3. On construira un nouveau bâtiment. → _______________
4. Ils avaient annulé la réunion. → _______________
5. On ouvrirait le magasin à 9h. → _______________

### Answers

1. Le projet **est approuvé** par le chef.
2. Le travail **a été terminé** par l'équipe.
3. Un nouveau bâtiment **sera construit**.
4. La réunion **avait été annulée**.
5. Le magasin **serait ouvert** à 9h.

---

## Practice 2: Transform to active

**Transform to active voice:**

1. La lettre a été envoyée par la secrétaire.  
   → _______________

2. Cette décision sera prise par le conseil.  
   → _______________

3. Le problème a été résolu par l'équipe technique.  
   → _______________

4. Les résultats avaient été annoncés par le directeur.  
   → _______________

### Answers

1. La secrétaire **a envoyé** la lettre.
2. Le conseil **prendra** cette décision.
3. L'équipe technique **a résolu** le problème.
4. Le directeur **avait annoncé** les résultats.

---

## Practice 3: Complete the news report

**Fill with passive forms:**

Un accident _____________ (1. se produire) hier sur l'autoroute E40. Trois personnes _____________ (2. blesser) et _____________ (3. transporter) à l'hôpital.

L'autoroute _____________ (4. fermer) pendant deux heures. La circulation _____________ (5. dévier) par la route nationale. Une enquête _____________ (6. ouvrir) par la police.

### Answers

1. s'**est produit** (pronominal, not really passive but similar)
2. **ont été blessées**
3. **ont été transportées**
4. **a été fermée**
5. **a été déviée**
6. **a été ouverte**

---

## Practice 4: Professional context

**Rewrite in passive for a formal report:**

1. L'entreprise a créé 50 emplois.  
   → _______________

2. Le directeur signera le contrat demain.  
   → _______________

3. Notre équipe a développé ce logiciel.  
   → _______________

4. Le gouvernement va adopter de nouvelles mesures.  
   → _______________

### Answers

1. 50 emplois **ont été créés** par l'entreprise.
2. Le contrat **sera signé** demain par le directeur.
3. Ce logiciel **a été développé** par notre équipe.
4. De nouvelles mesures **vont être adoptées** par le gouvernement.

## Practice 5: Mixed exercise

**Choose active or passive as appropriate:**

1. (News about election)  
   ○ Les citoyens ont élu le nouveau maire.  
   ○ Le nouveau maire a été élu.

2. (Reporting a crime with unknown actor)  
   ○ Quelqu'un a volé le tableau.  
   ○ Le tableau a été volé.

3. (Minutes of a meeting)  
   ○ Le comité a approuvé le budget.  
   ○ Le budget a été approuvé.

4. (Story about a person)  
   ○ Marie a écrit ce livre.  
   ○ Ce livre a été écrit par Marie.

### Best choices

1. **Le nouveau maire a été élu.** (passive - more objective)
2. **Le tableau a été volé.** (passive - unknown actor)
3. **Le budget a été approuvé.** (passive - formal)
4. **Marie a écrit ce livre.** (active - person is important)

## Practice 6: Alternative to passive

### "ON" as an alternative

| Passive | With "on" |
|:--------|:----------|
| Le français est parlé ici | **On** parle français ici |
| Ce vin est bu frais | **On** boit ce vin frais |

**Transform using "on":**

1. Les gaufres sont vendues partout. → _______________
2. Le néerlandais est parlé en Flandre. → _______________
3. Les documents seront envoyés demain. → _______________

### Answers

1. **On** vend des gaufres partout.
2. **On** parle néerlandais en Flandre.
3. **On** enverra les documents demain.
