# Passé Composé vs Imparfait : Le Guide Ultime

Comprendre la différence entre le **Passé Composé** et l'**Imparfait** est essentiel pour raconter des histoires en français. Voici comment savoir lequel utiliser.

## 1. La Règle d'Or

Imaginez une pièce de théâtre :
- **L'Imparfait** plante le décor (la scène, l'ambiance, les descriptions).
- **Le Passé Composé** indique les actions qui font avancer l'histoire.

## 2. Tableau Comparatif

| Usage | Passé Composé | Imparfait |
| :--- | :--- | :--- |
| **Fonction** | Actions principales, événements ponctuels | Arrière-plan, descriptions, habitudes |
| **Durée** | Limitée, précise, terminée | Indéfinie, continue |
| **Questions** | _"Qu'est-ce qui s'est passé ?"_ | _"Comment c'était ?"_, _"Que se passait-il ?"_ |
| **Mots-clés** | Soudain, tout à coup, hier, une fois | Souvent, tous les jours, d'habitude, pendant que |

## 3. Exemples Concrets

### Description vs Action
*   **Imparfait** : "Il **faisait** beau, les oiseaux **chantaient**..." (Le décor)
*   **Passé Composé** : "...soudain, un orage **a éclaté**." (L'action qui change tout)

### Habitude vs Événement Unique
*   **Imparfait** : "Quand j'étais jeune, je **jouais** au tennis tous les dimanches." (Habitude)
*   **Passé Composé** : "Dimanche dernier, j'**ai joué** contre mon frère." (Une fois spécifique)

### Action interrompue
*   **Imparfait** : "Je **dormais** paisiblement..." (Action en cours)
*   **Passé Composé** : "...quand le téléphone **a sonné** !" (Interruption)

## 4. Astuce Visuelle

Imaginez une ligne du temps :
- L'Imparfait est une ligne ondulée ~~~~~~~~ (ça dure, c'est flou).
- Le Passé Composé est un point précis ● (c'est arrivé à un moment T).

---

## 5. Pratique

Complétez :
"Hier soir, je (regarder) \_\_\_\_\_\_ la télé quand ma mère (appeler) \_\_\_\_\_\_."

*Réponse :*
"Hier soir, je **regardais** (action en cours) la télé quand ma mère **a appelé** (action soudaine)."
