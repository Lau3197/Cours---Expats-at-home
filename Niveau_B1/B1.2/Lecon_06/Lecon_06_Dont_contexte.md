# Leçon 06 : DONT en contexte

**Niveau** : B1.2  
**Module** : Module 3 : La chose dont on parle
**Type** : Consolidation  
**Durée estimée** : 2-3 heures  
**Objectifs** : Practice DONT in various contexts

---

## Review: DONT

**DONT replaces DE + noun**

| Use | Example |
|:----|:--------|
| Verbs with de | Le livre dont j'ai besoin |
| Possession | L'homme dont la femme travaille ici |
| Adjectives with de | Le résultat dont je suis fier |

---

## Practice 1: Complete with DONT

1. C'est le projet ___ je t'ai parlé.
2. Voici l'homme ___ les enfants sont médecins.
3. C'est le travail ___ je suis le plus fier.
4. C'est le film ___ j'ai envie de voir.
5. Voici la collègue ___ je m'occupe du dossier.

### Answers

1. dont 2. dont 3. dont 4. dont 5. dont

---

## Practice 2: QUI, QUE, OÙ or DONT?

1. L'homme ___ je vois est grand.
2. L'homme ___ je parle est grand.
3. L'homme ___ parle est grand.
4. L'endroit ___ je travaille est agréable.
5. Le livre ___ j'ai besoin est épuisé.

### Answers

1. **que** (direct object)
2. **dont** (parler de)
3. **qui** (subject)
4. **où** (place)
5. **dont** (avoir besoin de)

---

## Practice 3: Combine sentences

1. C'est un restaurant. Tout le monde parle de ce restaurant.
2. C'est une femme. Le mari de cette femme est avocat.
3. C'est un voyage. Je me souviens bien de ce voyage.
4. C'est un projet. Je suis responsable de ce projet.

### Answers

1. C'est un restaurant **dont** tout le monde parle.
2. C'est une femme **dont** le mari est avocat.
3. C'est un voyage **dont** je me souviens bien.
4. C'est un projet **dont** je suis responsable.
