---

## Review: All Hypothesis Types

| Type | Si clause | Result clause | Use |
|:-----|:----------|:--------------|:----|
| 1 | Présent | Futur | Real/Possible |
| 2 | Imparfait | Cond. présent | Unreal present |
| 3 | PQP | Cond. passé | Unreal past |
| Mixed | PQP | Cond. présent | Past → Present |

---

## Practice 1: Identify and complete

1. Si tu ___ (venir) demain, on ira au cinéma. → Type ___
2. Si j'___ (avoir) le temps, je voyagerais. → Type ___
3. Si elle ___ (accepter) hier, elle serait là. → Type ___
4. Si nous ___ (étudier) plus jeunes, nous serions bilingues maintenant. → Type ___

### Answers

1. **viens** / Type 1
2. **avais** / Type 2
3. **avait accepté** / Type 3
4. **avions étudié** / Mixed

---

## Practice 2: Transform between types

**Change from Type 2 to Type 3:**

1. Si j'avais le temps, je voyagerais.
   → Si j'___ le temps, j'___

2. Si elle était là, je serais content.
   → Si elle ___ là, j'___

### Answers

1. Si j'**avais eu** le temps, j'**aurais voyagé**.
2. Si elle **avait été** là, j'**aurais été** content.

## Practice 4: Mixed scenarios

**Choose the correct structure:**

1. (Past action → present result) Si tu ___ (prendre) le train, tu ___ (être) déjà arrivé.
2. (Real possibility) Si elle ___ (téléphoner), je lui ___ (répondre).
3. (Unreal present) Si nous ___ (avoir) plus d'argent, nous ___ (déménager).
4. (Unreal past) Si vous ___ (m'écouter), vous ___ (ne pas avoir) ce problème.

### Answers

1. avais pris / serais
2. téléphone / répondrai
3. avions / déménagerions
4. m'aviez écouté / n'auriez pas eu

---

## Grammar: Complete Summary

| Condition | Result | Structure |
|:----------|:-------|:----------|
| Real (now/future) | Certain | si + présent → futur |
| Unreal (now) | Would happen | si + imparfait → cond. présent |
| Unreal (past) | Would have happened | si + PQP → cond. passé |
| Unreal (past) | Would be now | si + PQP → cond. présent |

**Remember: NEVER use conditional after SI!**
