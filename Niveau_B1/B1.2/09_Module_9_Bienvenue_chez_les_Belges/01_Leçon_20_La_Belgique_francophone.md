## La Belgique francophone

### Géographie linguistique

La Belgique est un pays multilingue avec trois régions principales :

| Région | Langue(s) principale(s) |
|:-------|:------------------------|
| Flandre | Néerlandais |
| Wallonie | Français |
| Bruxelles-Capitale | Français & Néerlandais |
| Communauté germanophone | Allemand |

### Les provinces wallonnes

| Province | Chef-lieu |
|:---------|:----------|
| Hainaut | Mons |
| Liège | Liège |
| Luxembourg | Arlon |
| Namur | Namur |
| Brabant wallon | Wavre |

## Particularités du français belge

### Expressions typiques

| Belge | Français standard | Sens |
|:------|:------------------|:-----|
| septante | soixante-dix | 70 |
| nonante | quatre-vingt-dix | 90 |
| une fois | (particule d'insistance) | "Viens une fois !" |
| à tantôt | à tout à l'heure | See you later |
| s'il vous plaît | voilà | Here you go |
| drache | pluie forte | heavy rain |

### Vocabulaire spécifique

| Belge | Français standard |
|:------|:------------------|
| GSM | téléphone portable |
| kot | chambre d'étudiant |
| ring | périphérique |
| navetteur | banlieusard |
| pistolet | petit pain |

## Culture et vie quotidienne

### Gastronomie

| Spécialité | Description |
|:-----------|:------------|
| Moules-frites | Plat national |
| Gaufres | De Liège ou de Bruxelles |
| Chocolat | Renommée mondiale |
| Bières spéciales | Chimay, Orval, Westmalle... |
| Carbonnade flamande | Ragoût à la bière |
| Waterzooi | Plat gantois (poulet/poisson) |

### Traditions

| Événement | Description |
|:----------|:------------|
| Carnaval de Binche | Patrimoine UNESCO (Gilles) |
| Ducasse d'Ath | Géants processionnels |
| Festivals | Tomorrowland, Rock Werchter |

## Compréhension

**Questions:**

1. Quelles langues parle-t-on en Belgique ?
2. Comment dit-on "70" et "90" en Belgique ?
3. Qu'est-ce qu'un "kot" ?
4. Quelle est la spécialité culinaire nationale ?
5. Quand est la fête nationale belge ?

### Answers

1. Français, néerlandais, allemand
2. Septante, nonante
3. Une chambre d'étudiant
4. Les moules-frites
5. Le 21 juillet
