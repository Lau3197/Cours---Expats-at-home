# Leçon 16 : Chercher un appartement (Comparatifs & Adverbes)

**Niveau** : A2.1  
**Durée estimée** : 2-3 heures  
**Objectifs** : Comparer des logements, décrire la manière d'agir (adverbes)

---

## 1. Dialogue : La visite immobilière

**L'agent immobilier montre un appartement à Tom.**

> **Agent :** Visitez **librement**. Voici le salon. Il est **plus lumineux que** l'autre appartement, n'est-ce pas ?
> 
> **Tom :** Oui, c'est vrai. Et il est **mieux** situé. Mais le loyer est **plus cher**...
> 
> **Agent :** C'est normal, nous sommes dans le centre. Ici, les prix montent **rapidement**.
> 
> **Tom :** Il y a **autant de** chambres que dans l'autre ?
> 
> **Agent :** Oui, deux chambres. Mais celle-ci est **plus calmement** exposée, elle donne sur le jardin.
> 
> **Tom :** Je dois décider **sérieusement**. J'ai **moins de** budget que prévu.
> 
> **Agent :** Réfléchissez **tranquillement**. Mais cet appartement partira **vite** !

## 2. Grammaire : Les Adverbes en "-ment"

Pour décrire **comment** on fait une action, on utilise un adverbe.
*Exemple : Il marche. Comment ? → Il marche **rapidement**.*

### A. La Formation (Règle des 3 étapes)

1. Prenez l'adjectif au **masculin** : *Heureux*
2. Mettez-le au **féminin** : *Heureuse*
3. Ajoutez **-ment** : *Heureusement*

**Exemples :**
- Lent → Lente → **Lentement**
- Doux → Douce → **Doucement**
- Sérieux → Sérieuse → **Sérieusement**
- Actif → Active → **Activement**

> **Cas particuliers :**
> - Si l'adjectif masculin finit par une voyelle : Vrai → **Vraiment**, Poli → **Poliment** (On ajoute -ment direct).
> - Rapide → **Rapidement** (finit par 'e', on ajoute juste -ment).

### B. La Place de l'Adverbe

En général, l'adverbe se place **APRÈS LE VERBE**.

- Je visite **rapidement**.
- Elle répond **poliment**.
- Il cherche **activement**.

## 3. Pratique 1 : Créez les adverbes

**Transformez l'adjectif en adverbe :**

1. (Calme) Il parle _______________.
2. (Sérieux) Elle travaille _______________.
3. (Grand) Il ouvre _______________ la porte.
4. (Tranquille) Nous marchons _______________.
5. (Clair) Il explique _______________ la situation.

### Réponses

1. Il parle **calmement**.
2. Elle travaille **sérieusement**. (Sérieux -> Sérieuse -> Sérieusement)
3. Il ouvre **grandement** la porte.
4. Nous marchons **tranquillement**.
5. Il explique **clairement** la situation.

## 4. Grammaire : Les Trois Types de Comparaison

On peut comparer trois choses : des **Adjectifs** (qualités), des **Adverbes** (manières), et des **Noms** (quantités).

### A. Comparer des Adjectifs et des Adverbes (Qualité/Manière)
*C'est la même structure.*

| Comparaison | Structure | Exemple (Adjectif) | Exemple (Adverbe) |
| :--- | :--- | :--- | :--- |
| **+** (Plus) | **plus** ... **que** | Il est **plus** grand **que** toi. | Il court **plus** vite **que** toi. |
| **-** (Moins) | **moins** ... **que** | C'est **moins** cher **que** ça. | Je mange **moins** lentement **que** lui. |
| **=** (Égal) | **aussi** ... **que** | Elle est **aussi** gentille **que** Marie. | Il parle **aussi** poliment **que** son frère. |

> **Attention ! Irréguliers :**
> - **Bon** (adj) devient **Meilleur** (Ex: Ce gâteau est **meilleur**).
> - **Bien** (adv) devient **Mieux** (Ex: Il chante **mieux**).

### B. Comparer des Noms (Quantité)
*On ajoute "DE" après le comparatif.*

| Comparaison | Structure | Exemple |
| :--- | :--- | :--- |
| **+** (Plus) | **plus de** ... **que** | J'ai **plus de** chance **que** toi. |
| **-** (Moins) | **moins de** ... **que** | Il y a **moins de** bruit **que** hier. |
| **=** (Égal) | **autant de** ... **que** | Il y a **autant de** chambres **que** de salles de bain. |

## 5. Pratique 2 : Adjectif, Adverbe ou Nom ?

**Complétez avec le bon comparatif :**

1. (Grand / +) Cet appartement est _________________ l'autre.
2. (Lumière / -) Il y a _________________ dans le couloir.
3. (Vite / =) Tu marches _________________ moi ?
4. (Meublé / +) Le studio est _________________ la chambre.
5. (Placards / =) Il y a _________________ dans la cuisine que dans l'entrée.

### Réponses

1. Cet appartement est **plus grand que** l'autre. (Adjectif)
2. Il y a **moins de lumière** dans le couloir. (Nom → moins DE)
3. Tu marches **aussi vite que** moi ? (Adverbe)
4. Le studio est **plus meublé que** la chambre. (Adjectif)
5. Il y a **autant de placards** dans la cuisine que dans l'entrée. (Nom → Autant DE)

## 6. Vocabulaire : Le Logement

| Mot | Définition |
| :--- | :--- |
| **Le loyer** | L'argent qu'on paie chaque mois. |
| **Les charges** | L'eau, l'électricité, l'entretien (souvent en plus du loyer). |
| **La caution** | L'argent bloqué comme garantie. |
| **Un état des lieux** | Vérifier l'appartement avant d'entrer. |
| **Meublé / Non-meublé** | Avec ou sans meubles (lits, tables...). |
| **Le préavis** | Le délai avant de partir (ex: 3 mois). |

## 7. Mise en Pratique Complète

**Comparez l'Appartement A et l'Appartement B :**

| | Appartement A | Appartement B |
| :--- | :--- | :--- |
| **Prix** | 800€ | 650€ |
| **Taille** | 50m2 | 50m2 |
| **Calme** | Très calme | Un peu bruyant |
| **Chambres** | 2 | 1 |

1. *(Prix / cher)* : L'appartement A est ____________________ l'appartement B.
2. *(Taille / grand)* : L'appartement A est ____________________ l'appartement B.
3. *(Calme)* : L'appartement A est situé ____________________ (calmement) l'appartement B.
4. *(Chambres)* : L'appartement A a ____________________ chambres ____ l'appartement B.

### Réponses
1. L'appartement A est **plus cher que** l'appartement B.
2. L'appartement A est **aussi grand que** l'appartement B.
3. L'appartement A est situé **plus calmement que** l'appartement B.
4. L'appartement A a **plus de chambres que** l'appartement B.

## 8. Récapitulatif et Erreurs à éviter

### Ce qu'il faut retenir
1. **Adverbes en -ment** : Féminin de l'adjectif + ment (Lente → Lentement).
2. **Comparatifs** :
   - Qualité : *Plus/Moins/Aussi* + Adjectif + *que*
   - Quantité : *Plus/Moins/Autant* + **DE** + Nom + *que*

### Erreurs Fréquentes

| Incorrect (X) | Correct (V) |
| :--- | :--- |
| C'est plus bon. | C'est **meilleur**. |
| Il parle plus bien. | Il parle **mieux**. |
| Il a autant livres que moi. | Il a **autant de** livres que moi. (N'oubliez pas le **DE** avec les noms !) |
| Elle est plus petite que lui. | Elle est **plus petite que** lui. (N'oubliez pas le **QUE**) |
