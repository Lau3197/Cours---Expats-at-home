# Leçon 09 : Hypothèses au passé (Consolidation)

**Niveau** : B1.1  
**Type** : Consolidation  
**Durée estimée** : 2-3 heures  
**Objectifs** : Master all three types of hypotheses

---

## Review: Types of Hypotheses

### Type 1: Real/Possible (present/future)

| French | English |
|:-------|:--------|
| Si + présent, futur | If ..., will... |
| Si tu viens, je serai content | If you come, I'll be happy |

### Type 2: Unreal Present (A2)

| French | English |
|:-------|:--------|
| Si + imparfait, conditionnel présent | If ..., would... |
| Si j'avais le temps, je voyagerais | If I had time, I would travel |

### Type 3: Unreal Past (B1) - NEW!

| French | English |
|:-------|:--------|
| Si + plus-que-parfait, conditionnel passé | If had ..., would have... |
| Si j'avais su, j'aurais agi | If I had known, I would have acted |

---

## Practice 1: Identify the type

**Which type of hypothesis?**

1. Si tu viens demain, on ira au cinéma. → ___
2. Si j'étais riche, j'achèterais une maison. → ___
3. Si j'avais étudié, j'aurais réussi. → ___
4. Si elle avait le temps, elle voyagerait. → ___
5. Si nous étions partis plus tôt, nous serions arrivés à l'heure. → ___

### Answers

1. Type 1 (Real/possible)
2. Type 2 (Unreal present)
3. Type 3 (Unreal past)
4. Type 2 (Unreal present)
5. Type 3 (Unreal past)

---

## Practice 2: Choose the correct tenses

**Complete with the appropriate tenses:**

1. Si tu _____ (venir), je _____ (être) content. (tomorrow)
2. Si je _____ (avoir) de l'argent, je _____ (acheter) une voiture. (now, but I don't)
3. Si elle _____ (étudier), elle _____ (réussir). (past, but she didn't)
4. Si nous _____ (savoir), nous _____ (prévenir) tout le monde. (past)
5. Si il _____ (faire) beau demain, nous _____ (aller) à la plage. (possible)

### Answers

1. Si tu **viens**, je **serai** content. (Type 1)
2. Si j'**avais** de l'argent, j'**achèterais** une voiture. (Type 2)
3. Si elle **avait étudié**, elle **aurait réussi**. (Type 3)
4. Si nous **avions su**, nous **aurions prévenu** tout le monde. (Type 3)
5. S'il **fait** beau demain, nous **irons** à la plage. (Type 1)

## Practice 3: Transform between types

**Change from Type 2 to Type 3:**

1. Si j'avais le temps, je voyagerais.  
   → _______________

2. Si tu étais là, je serais content.  
   → _______________

3. Si nous avions plus d'argent, nous déménagerions.  
   → _______________

### Answers

1. Si j'**avais eu** le temps, j'**aurais voyagé**.
2. Si tu **avais été** là, j'**aurais été** content.
3. Si nous **avions eu** plus d'argent, nous **aurions déménagé**.

## Practice 4: Mixed hypotheses with present result

**Sometimes past conditions affect present situations:**

| Past condition | Present result |
|:---------------|:---------------|
| Si + PQP | Conditionnel présent |
| Si j'avais étudié | je serais médecin (aujourd'hui) |

**Complete:**

1. Si j'avais fait du sport quand j'étais jeune, je _____ (être) en meilleure forme maintenant.
2. Si elle avait accepté ce travail, elle _____ (vivre) à Paris.
3. Si nous avions investi, nous _____ (avoir) plus d'argent aujourd'hui.

### Answers

1. je **serais** en meilleure forme maintenant.
2. elle **vivrait** à Paris.
3. nous **aurions** plus d'argent aujourd'hui.

## Grammar: Complete Hypothesis Summary

| 3b | Plus-que-parfait | Cond. présent | Si j'avais étudié, je serais médecin |

## Common Mistakes Review

### Never use conditional after SI

| Incorrect | Correct |
|:----------|:--------|
| Si j'aurais su | Si j'**avais** su |
| Si j'aurais voulu | Si j'**avais** voulu |
